## Comandos a ejecutar
1: ```composer install```

2: ```npm install```
3: ```cp .env.example .env```

4: ```php artisan key:generate```

5:```php artisan serve```

6: para ejecutar sass (estilos) ```npm run dev``` o ```npm run watch```
